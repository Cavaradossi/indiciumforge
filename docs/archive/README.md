# Archive

Historical, migration-era working documents retained for provenance and audit. Files here
are **not** a source of truth for current capabilities and are no longer actively updated.

For current state, use the live documents instead:

- Capability status: [CAPABILITY_REGISTER.md](../../CAPABILITY_REGISTER.md)
- Release history: [RELEASE_NOTES.md](../../RELEASE_NOTES.md)
- Forward roadmap: [MIGRATION_ROADMAP.md](../MIGRATION_ROADMAP.md)

Permanent design boundaries that originated during migration (for example the
anti-inheritance rules in
[ADR-0019](../decisions/ADR-0019-anti-inheritance-from-indiciumgrid-v0.9.md)) remain in
`docs/decisions/` because they are still enforced by the security test suite, not archived
here.
